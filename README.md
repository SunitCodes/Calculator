# Calculator
A simple calculator for beginners <br>
This is a simple calculator web application built using HTML, CSS, and JavaScript.
<br><br>

Features
<br>
Addition, subtraction, multiplication, and division operations.<br>
Clear button to reset the calculator.<br>
Keyboard support for basic operations.<br>

<br>
Usage<br>
Click on the buttons to input numbers and perform operations.<br>
Use the keyboard (numeric keys and operators) for input and operations as well.<br>
Press the "C" button to clear the input.<br>
